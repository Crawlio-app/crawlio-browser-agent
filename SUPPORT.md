# Support

## How to Get Help

Crawlio Agent is an open-source project. We do our best to respond to issues in a timely manner, but please understand this is not a commercial support channel.

### Bug Reports & Feature Requests

Use [GitHub Issues](https://github.com/Crawlio-app/crawlio-browser-agent/issues) to report bugs or request features. Please use the issue templates provided.

Before opening a new issue:
1. Search [existing issues](https://github.com/Crawlio-app/crawlio-browser-agent/issues) to avoid duplicates
2. Include your crawlio-browser version (`npx crawlio-browser --version`)
3. Include your environment (OS, Node.js version, Chrome version, MCP client)

### Documentation

- [Getting Started](https://www.crawlio.app/browser-agent)
- [Architecture Guide](https://docs.crawlio.app/browser-agent/jit-context-runtime)

### What This Repo is NOT

- A general help forum for MCP or Chrome extensions
- A place to ask questions about unrelated browser automation tools

If you have a question about MCP itself, see the [Model Context Protocol docs](https://modelcontextprotocol.io).
