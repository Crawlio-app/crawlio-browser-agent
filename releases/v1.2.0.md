# v1.2.0

**Date:** 2026-03-01

## Changes

### Added

- `npx crawlio-agent init` — interactive wizard that auto-detects environment and configures the right transport (stdio, `.mcp.json`, or portal)
- New flags: `--portal`, `--yes` / `-y`, `-a <agent>`, `--dry-run`
- `alarms` permission for WebSocket reconnect scheduling
- GitHub Actions CI pipeline (`.github/workflows/ci.yml`) — Node 18/20/22
- `prepublishOnly` script for publish gate

### Removed

- `nativeMessaging` from `optional_permissions`
- `commander` from dependencies
- `idb` from dependencies — only used by extension code

### Fixed

- Node path resolution — `process.execPath` replaced with `resolveNodePath()`
- Windows path handling — `fileURLToPath()` instead of `new URL().pathname`
- Windows npx — platform-aware `npx.cmd` detection
- Cross-platform skill discovery — `readdirSync` instead of `find`
- Port conflict detection with clear error message
