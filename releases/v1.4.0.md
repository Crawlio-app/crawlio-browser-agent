# v1.4.0

**Date:** 2026-03-02

## Changes

### Added

- **Session recording** — 3 new MCP tools (`start_recording`, `stop_recording`, `get_recording_status`) that capture a full browser session as structured data
- **Recording compiler** — generates replayable SKILL.md files from captured interactions
- 12 interaction tools automatically intercepted during recording
- `compileRecording` exposed in the execute sandbox for capture-and-compile in a single call
- Multi-instance MCP server with dynamic port allocation (9333–9342) — run up to 10 agents in parallel
- 72 new recording tests across 3 test files

### Changed

- **`--cloudflare` flag for init wizard** — `npx crawlio-browser init --cloudflare` adds Cloudflare MCP integration (89 tools) with zero wrangler dependency
- Replaced `mcp-remote` with `@cloudflare/mcp-server-cloudflare` — eliminates broken browser OAuth flow

### Fixed

- Recording tool error handling — all 3 recording tool handlers now catch `bridge.send()` errors internally
