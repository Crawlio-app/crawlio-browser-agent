# Browser Agent v1.6.1

Released: 2026-03-13

## New Features

- **114 tools** (was 100) — 14 new tools across detection, SEO, and tracking analysis
- **Detection wedge**: parse Facebook/GA4/TikTok/LinkedIn/Pinterest tracking pixels, validate against vendor parameter schemas, inspect runtime dataLayer/fbq/gtag state, detect duplicate pixel fires
- **SEO intelligence**: 11-dimension on-page SEO audit, raw-vs-rendered comparison, CrUX field metrics, SERP overlay, robots.txt and sitemap validation
- **Technology detection**: Wappalyzer-style fingerprinting with additive confidence scoring and implies/excludes relationship graph (~200 technologies)
- **147 searchable commands** in code mode (was 133)
- **17 smart.\* higher-order methods** (was 8) — added parseTrackingPixels, validateTracking, inspectDataLayer, detectDuplicates, detectTechnologies, diffSnapshots, and more

## Improvements

- Hardened tool counts across README, package.json, init wizard, and execute tool description
- Updated npm description with new feature categories

## Fixes

- `parseInt` masking zero values in fingerprint-db confidence scoring
- Greedy regex in version ternary resolution eating colons
- User-controlled parameter values truncated in validation error messages (CWE-532)
- Levenshtein distance capped at 256 chars to prevent DoS
- Tracking heuristic regex `[?/s]` corrected to `s?[?/]`
- `tabSerpState` migration added to `tabs.onReplaced` handler
